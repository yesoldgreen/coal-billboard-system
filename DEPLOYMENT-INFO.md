## 🔧 技术架构

- **后端**: Node.js + Express + SQLite
- **前端**: 纯HTML + CSS + JavaScript
- **认证**: JWT
- **数据库**: SQLite3
- **部署**: 云服务器 + 端口导出

---

## 📝 注意事项

1. **数据持久化**: 数据存储在SQLite数据库中，系统重启后数据不会丢失
2. **自动刷新**: 客户端每30秒自动刷新数据
3. **无历史记录**: 系统不保存历史数据，每次更新会覆盖之前的数据
4. **公网访问**: 当前URL为临时预览地址，建议绑定正式域名
