## 📋 快速使用指南

### 1️⃣ 登录后台管理系统

1. 访问：https://3000-capy-1772251385994-336037-preview.happycapy.ai/admin/index.html
2. 输入用户名：`hulianshikong`
3. 输入密码：`hlsk2026`
4. 点击"登录"

### 2️⃣ 管理告示牌

**创建新告示牌**:
- 点击"创建告示牌"按钮
- 输入告示牌名称（如：汇能长滩 煤矿内参）
- 点击确定

**修改告示牌名称**:
- 找到要修改的告示牌
- 点击"改名"按钮
- 输入新名称

**删除告示牌**:
- 找到要删除的告示牌
- 点击"删除"按钮
- 确认删除

**复制客户端链接**:
- 点击"📋 复制客户端链接"
- 链接会自动复制到剪贴板
- 分享给需要查看的用户

### 3️⃣ 编辑数据

**进入编辑页面**:
- 点击告示牌的"编辑数据"按钮
- 进入数据编辑器

**编辑排队拉运数据**:
1. 在"排队拉运"标签页
2. 点击"+ 添加行"添加数据行
3. 填写：坑口、已磨约、排队中、已叫号、已入场
4. 可选：添加广告（点击"+ 添加广告"）
5. 点击"💾 保存排队拉运数据"

**编辑质量/价格数据**:
1. 切换到"质量/价格"标签页
2. 点击"+ 添加行"添加数据行
3. 填写：坑口、热值、灰分、硫、含税价格、较上期变动
4. 变动值格式：`+10` 或 `-5`
5. 可选：添加广告
6. 点击"💾 保存质量/价格数据"

**编辑物流费用数据**:
1. 切换到"物流费用"标签页
2. 点击"+ 添加行"添加数据行
3. 选择类型：
   - **直运**: 填写起点、终点、运费
   - **集运站**: 填写集运站名称、站运费、站台费
4. 可选：添加广告
5. 点击"💾 保存物流费用数据"

### 4️⃣ 查看客户端效果

**方式1: 从后台复制链接**
- 在告示牌列表中点击"📋 复制客户端链接"
- 在新窗口打开链接

**方式2: 直接访问**
- 访问：https://3000-capy-1772251385994-336037-preview.happycapy.ai/client/index.html?id=1
